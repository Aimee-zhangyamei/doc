    4.git add #

    语法 效果
    git add -A 从工作区填加所有的改变的文件到暂存区
    git add . 从工作区填加新的文件和改变的文件到暂存区，不包括删掉的文件。
    git add -u 从工作区填加 编辑过和删除的文件到暂存区，不包括新的文件。
